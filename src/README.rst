************
Installation
************

Requirements
============

- Python 3.10.19
- pip 20.0.2

Installing by using PyPi
========================

Install
-------
.. code-block:: bash

	git clone https://github.com/VoytRuslan/M1P.git /tmp/ProjectTemplate
	python3 -m pip install /tmp/ProjectTemplate/src/

Uninstall
---------
.. code-block:: bash

  python3 -m pip uninstall mylib
